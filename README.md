# CNWPer SEO Tags - 自动内链关键字插件

<p align="center">
  <strong>WordPress 自动内链与 SEO 关键字管理利器</strong>
</p>

一款专为 WordPress 设计的 SEO 增强插件，能够自动化管理网站关键字标签（TAGS），实现批量关键字内链、智能匹配文章内容，有效提升网站内部链接结构和搜索引擎优化效果。

---

## ✨ 核心功能

### 1. 自动添加 TAGS 标签
- **预设关键字库**：在后台维护一份关键字列表，支持一行一个或空格分隔
- **发布即添加**：新建或更新文章时，插件自动从关键字库中随机选取相关标签并添加到文章
- **智能去重**：每篇文章仅执行一次自动添加，避免重复处理

### 2. 自动内链（TAGS 内链）
- **一键开启**：将文章内容中的 TAGS 关键字自动转换为指向对应标签归档页的链接
- **智能替换**：自动识别并替换正文中的关键字，每个关键字首次出现处添加链接
- **安全过滤**：自动跳过已有链接、图片等 HTML 标签内的文字，避免破坏页面结构

### 3. 批量部署关键字
- **灵活范围**：支持按文章 ID 批量操作，格式支持：
  - 逗号分隔：`1,3,4,10`
  - 区间范围：`10-100`
- **独立配置**：批量任务可单独设置频率、匹配规则，不影响主设置
- **操作日志**：记录每次批量执行的详情，便于追溯与排查

### 4. 智能匹配模式
- **匹配标题**：仅当文章标题包含关键字时才添加该 TAGS，提高相关性
- **匹配内容**：仅当文章正文包含关键字时才添加该 TAGS，避免无关标签
- **可组合使用**：两种匹配模式可同时开启，实现更精准的标签分配

### 5. 防作弊与数量控制
- **单篇上限**：限制每篇文章的 TAGS 数量（默认 9 个），降低被搜索引擎误判为作弊的风险
- **频率控制**：可设置每篇文章添加的关键字数量，支持固定值（如 `3`）或范围（如 `3-6`）随机

---

## 📦 安装方法

1. 将插件文件夹 `cnwper-seo-tags` 上传至 `/wp-content/plugins/` 目录
2. 在 WordPress 后台 **插件** 列表中激活插件
3. 在左侧菜单进入 **自动关键字设置** 进行配置

---

## ⚙️ 配置说明

### 主设置页面（自动关键字设置）

| 选项 | 说明 |
|------|------|
| **开启** | 总开关，关闭后不会自动为文章添加 TAGS |
| **自动内链** | 是否将正文中的 TAGS 关键字自动转换为内链 |
| **关键字** | 预设关键字列表，一行一个或空格分隔 |
| **频率** | 每篇文章插入的关键字数量，如 `3` 或 `3-6`（随机） |
| **匹配标题** | 仅当标题包含关键字时才添加 |
| **匹配内容** | 仅当正文包含关键字时才添加 |
| **单篇数量** | 每篇文章 TAGS 上限，建议 10 以内 |

### 批量部署工具

| 选项 | 说明 |
|------|------|
| **关键字组** | 本次批量使用的关键字，仅用于本次操作 |
| **文章范围** | 目标文章 ID，如 `1,2,9,11` 或 `10-100` |
| **频率** | 同主设置 |
| **匹配标题 / 匹配内容** | 同主设置 |
| **单篇数量** | 同主设置 |

---

## 📋 使用建议

1. **关键字数量**：建议单篇文章 TAGS 控制在 3–5 个，避免过度堆砌
2. **相关性优先**：开启「匹配标题」或「匹配内容」可提升标签与文章的相关性
3. **谨慎批量**：批量操作时注意数量和匹配规则，过多无关关键字可能影响 SEO
4. **稳定维护**：不建议频繁大批量更换预设关键字，保持一定稳定性更利于收录
